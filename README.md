
# 🛠️ WiNeT-Windows-Network-Toolkit

A lightweight, powerful, and automated Batch script designed to diagnose and fix common Windows networking issues with a single click. No need to remember complex CMD commands anymore!

---

## ✨ Features

*   **Auto-Elevation**: Automatically requests Administrative privileges to run system-level repairs.
*   **IP Management**: Easily show, release, and renew your IP configuration.
*   **DNS & Stack Reset**: Flush DNS cache and reset Winsock/TCP-IP stacks to fix connectivity drops.
*   **Quick Diagnostics**: Built-in Ping tests and Network Statistics (Netstat).
*   **One-Click Repair**: A "Full Repair" mode that runs all essential fixes in sequence.
*   **UI/UX**: Clean Cyan-on-Black terminal interface for a professional look.

## 🚀 How to Use

1.  **Download** the `Network_Toolkit.bat` file.
2.  **Double-click** the file to run it.
3.  Select an option from **1 to 12** based on your requirement.
4.  Follow the on-screen instructions (Some repairs might require a PC restart).

## 🛠️ Built With

*   **Language**: Windows Batch Script (.bat)
*   **Target OS**: Windows 10 / 11
